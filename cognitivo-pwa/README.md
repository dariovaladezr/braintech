# Entrena tu mente — Web App (PWA)

App de entrenamiento cognitivo familiar. Funciona en iPhone y Android sin instalar nada desde la App Store.

---

## Opción A — Subir a Netlify (recomendada, gratis, 2 minutos)

1. Ve a https://app.netlify.com y crea una cuenta gratuita
2. En el dashboard haz clic en "Add new site" → "Deploy manually"
3. Arrastra la carpeta **`build/`** a la página
4. Netlify te da un link tipo: https://nombre-random.netlify.app
5. **Comparte ese link con tu familia por WhatsApp**

### Instalar en iPhone (Safari):
1. Abre el link en Safari
2. Toca el botón de compartir (cuadro con flecha)
3. "Añadir a pantalla de inicio"
4. ¡Aparece como app en el iPhone!

### Instalar en Android (Chrome):
1. Abre el link en Chrome
2. Toca el menú (3 puntos)
3. "Añadir a pantalla de inicio"

---

## Opción B — Correr localmente (sin internet)

```bash
npm install
npm start
```
Abre http://localhost:3000 en el navegador.

---

## La carpeta `build/` ya está lista

Si ya tienes el proyecto, la carpeta `build/` contiene la app compilada lista para subir.
No necesitas instalar nada para subirla a Netlify — solo arrastra la carpeta.
